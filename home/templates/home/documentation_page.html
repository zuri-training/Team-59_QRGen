API Documentation

This is a light weight QR code service, which provides an easy way to generate, download and share QR code across different platform. An API (Application Programming Interface) is a protocol intended to be used as an interface by software components to communicate with each other. Our API supports many methods, so there should not be a problem coding cool applications.

If you find any bug, or have any questions, or any suggestions please get in touch with us.

Authentication

Our API use JWT (Json Web Tokens) for user authentication. Each request to a restricted endpoint shall require an 'Access (Bearer)' Header containing the user's access code. Hence, only registered users with active access codes can access these restricted endpoints.

Create QR Code

Request Method

POST

Request URL

/qrcodes/

Request Parameter

{
title: "YOUR_TITLE"
user_id: "YOUR_USER_ID"
type: "QR_CODE_TYPE"
input_url: "QR_CODE_CONTENT"
}

Response

{
title: "YOUR_TITLE"
user_id: "YOUR_USER_ID"
type: "QR_CODE_TYPE"
input_url: "QR_CODE_CONTENT"
}
